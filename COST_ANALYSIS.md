# LearnByTeach AI 비용 분석 보고서

작성: 2026-06-26 · 기준 모델: **gpt-4o (전 호출)** · 측정 방식: 실제 프롬프트를 o200k 토크나이저로 실측 + 완료 대화 1건(학생007, 39메시지) 재생(replay)

> 환율 ₩1,380/$ 가정. 가격: gpt-4o `$2.50/1M in, $10/1M out` (캐싱 입력 `$1.25`), gpt-4o-mini `$0.15/$0.60`. (2026 표준 가정 — 변동 가능)

---

## 1. 호출 1회당 입력 토큰 (실측)

| 호출 종류 | 입력 토큰 | 비고 |
|---|---|---|
| 시스템 프롬프트(대화 제외) | **2,043** | 그중 지식파일 911 + 정적 규칙 ~1,100 |
| 완료 심판(대화 20개 포함) | **2,791** | 완료기준 + 정답지 + 전체 대화 |
| 이해도 분석 | **1,489** | 정답지 + 상태 + 최근 6메시지 |

핵심: **지식파일·전체 대화가 거의 매 호출에 다시 실림.**

## 2. 대화 1건 비용 (= 학생 1명이 한 수업 3단계 완주, 39메시지)

호출 42회, 입력 누적 ~125K, 출력 ~6K 토큰 기준:

| 시나리오 | 비용 | 원화 |
|---|---|---|
| **gpt-4o (캐싱 없음)** | **$0.373** | **₩514** |
| gpt-4o (시스템 프롬프트 캐싱 가정) | $0.216 | ₩299 |
| gpt-4o-mini (참고) | $0.022 | ₩31 |

→ **4o는 mini 대비 약 16.7배.** 자동 프롬프트 캐싱이 걸리면 실비용은 ~42% 낮아짐.

## 3. 규모별 (학생 1명 = 한 수업 1회 완주 기준)

| 인원 | 4o (캐싱X) | 4o (캐싱) | mini |
|---|---|---|---|
| 30명 (한 반) | $11.2 (₩15,428) | $6.5 | $0.67 |
| 200명 (한 학교) | $74.5 (₩102,854) | $43.3 | $4.47 |
| 1,000명 | $372.7 (₩514,271) | $216.4 | $22.36 |

> 주의: "1회 완주" 기준. 학생이 여러 수업·여러 번 쓰면 그만큼 곱해짐.

## 4. 절약안 — 하이브리드 (채팅=mini, 심판·이해도=4o)

판정 정확도가 필요한 곳(심판·이해도)만 4o, 수다스러운 채팅은 mini:

| | 대화 1건 | 1,000명 |
|---|---|---|
| 전체 4o | $0.373 | $373 |
| **하이브리드** | **$0.181** | **$181** |

→ **약 52% 절감**, 그러면서 핵심(완료 판정·오개념 차단)은 4o 유지. 채팅 품질은 이미 mini로도 충분했음.

## 5. 지식파일 크기 민감도 (⚠️ 중요)

현재 테스트 지식파일은 911토큰으로 **작음**. 실제 지도서 기반이면 훨씬 큼. 지식파일은 시스템·심판·이해도 호출마다 재전송됨:

| 지식파일 크기 | 대화 1건 (4o) |
|---|---|
| x1 (911토큰, 현재) | $0.37 |
| x5 (4,555토큰) | $0.77 |
| x10 (9,110토큰) | $1.27 |

→ **지식파일을 키우면 비용이 거의 선형으로 늘어남.** 지도서 통째로 넣지 말고 핵심만 정제할 것.

## 6. 비용 동인 (큰 순서)

1. **전체 대화 재전송** — 매 호출마다 누적 대화를 다시 보냄 → 대화가 길수록 비용 제곱으로 증가
2. **심판 호출** — minMessages 이후 메시지마다 큰 호출 추가(정답지+전체 대화)
3. **지식파일** — 거의 모든 호출에 포함
4. **큰 시스템 프롬프트** — 정적 규칙 ~1,100토큰

## 7. 권장 최적화 (효과 큰 순)

1. **하이브리드 모델** — 채팅 mini / 심판·이해도 4o → ~52% 절감 (품질 손실 거의 없음)
2. **프롬프트 캐싱 활용** — 시스템 프롬프트를 안정적 접두부로 유지 (자동 적용, ~42% 절감). 1·2번 중첩 가능
3. **심판에 전체 대화 대신 최근 N개 또는 요약** 전달 → 심판 입력 대폭 축소
4. **심판 빈도 축소** — 매 메시지가 아니라 2~3메시지마다, 또는 AI가 "완료 신호" 줄 때만
5. **지식파일 정제** — 지도서 통째 X, 단원 핵심만
6. **오래된 대화 요약/절단** — 컨텍스트 길이 상한

## 결론

- 현재(전 호출 4o)는 **학생 1명 완주당 약 ₩300~500**, 한 반(30명) **약 ₩6,500~15,000** 수준.
- 가장 현실적 절감: **하이브리드(채팅 mini + 심판 4o) + 캐싱**으로 **70%+ 절감 가능**하면서 핵심 기능 유지.
- 가장 큰 리스크: **지식파일을 크게 넣는 것** + **긴 대화** — 둘 다 비용을 빠르게 키움.
